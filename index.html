About Me

The Builder Behind the Code

I'm Earshadul — a full-stack engineer with 7+ years of building and shipping web and mobile products end-to-end. Based in Dhaka, Bangladesh, I work with clients and companies across the US, India, and the Gulf region.

My lane is AI-powered application development — using Next.js, Node.js, and Python to build products where LLMs are at the core, not just a bolt-on feature. My flagship project, pencilerkali.com, is a fully autonomous AI news pipeline that scrapes global news, translates it to Bangla using LLMs, and auto-publishes to the site and Facebook — zero human intervention.

I deploy everything myself over SSH on Linux servers. I understand the full stack — from the database schema to the CDN edge — because I've had to build and fix all of it alone.

What makes me different: Most candidates chasing AI roles have read tutorials. I have autonomous, production AI systems running today — and I built them solo.
